# VTuberMusic-UWP

# 这是什么？
VTuberMusic 的 UWP 客户端，为 Windows 10 设备设计。

# 关于开发
## 环境搭建
操作系统: Windows 10 1809 及以上版本 ( 建议使用 Windows 10 1909 )  
开发工具: Visual Studio 2019 及以上版本  
SDK: "通用 Windows 开发" 工作负载 (可通过 "Visual Studio Installer" 安装)
